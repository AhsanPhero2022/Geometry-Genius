What is DOM and what is the purpose of DOM?

What is DOM?
DOM means (document object model). It is a programming interface for web documents. It works that how a web browser present a web page internally. It can catch HTML elements and change them.

Purpose of DOM: The DOM (document object model) defines a standrad for accessung document. It is a platform and Language netural interface that allows programs and script to dynamically access and update the content, stracture and style a document.

How will you select HTML elements using DOM? Name the DOM methods.

I can select HTML element using DOM methods.

This is DOM methods:
document.getElementById(). get an element by the ID attribute. document.getElementsByClassName(). get or select elements by their HTML tag name. document.querySelector(). get elements by their class name.
