What is event bubble?

The Event bubbling, also known as event propagation, is a concept in JS where events triggered on a nested or child element also trigger on its parent element. When an event occurs on an element, Event bubbling first object the handlers attached to that element, then its parent element, and more to bubble up the DOM tree until it reaches the Document object. This can be helpful for creating add event listeners that apply to multiple components.

What is a callback function and why will you use it?

In JavaScript, I use the callback function for that is passed as an argument to another function and executed inside that function. Callback functions are commonly used in asynchronous programming, where a function has to wait for an operation to complete before executing the rest of the code. in can can also be used to simplify code and make it perfect modular, as they allow me to pass behaviors as parameters to a function.
